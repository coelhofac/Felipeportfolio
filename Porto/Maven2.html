Small Market Diagnostic: Providing insights for sales increasing

Project made entirely in PowerBi, with PowerQuery, DAX, and Visualization Tools.

Results

Our goal in this project was provide an interactive dashboard with insights that could help diagnose the business' health and giving practical suggestions for achieving a better performance in critical points. The practical suggestions are in the presentation, you can get them here.

As you shall see many of our measures were'nt used, but by the sake of cleanliness of our report, but anyway, we have a project ready for alterations according to the requests of the stakeholders. Making a versatile project.



Importing and cleaning

I've choosed to stablish all the connections between the tables in the BI, what is more appropriate for bigger datasets
Our cleaning work can be summed up in: 1. Verifying and correcting each column's data type 2. Extracting the name of the month of the dates. 3. Identifying and correcting errors through column profiling Correcting capitalization and trimming extra spaces.


Stablishing relationships and creating measures

In this step we've focused on creating our data relationships, building the SCHEMA, stablishing primary and foreign keys, creating all the needed measures for data exploration and visualization based on PRIMARY keys.

Measures Formulas: 


                60-Day Revenue = CALCULATE([Total Revenue], DATESINPERIOD('Calendar'[date], MAX('Calendar'[date]), -60, DAY))
                
                CustomersBoughtLastYear = 
                CALCULATE(
                    DISTINCTCOUNT(Customers[customer_id]),
                    Calendar[Year] = YEAR(TODAY()) - 1
                )
            
                CustomersBoughtThisYear = 
                CALCULATE(
                    DISTINCTCOUNT(Customers[customer_id]),
                    Calendar[Year] = YEAR(TODAY())
                )
            
                DiminishedCustomers = 
                [TotalDistinctCustomers] - [CustomersBoughtThisYear]
            
                Last Month Returns = CALCULATE([Total Returns], DATEADD('Calendar'[date], -1, MONTH))
            
                Last Month Revenue = CALCULATE([Total Revenue], DATEADD('Calendar'[date], -1, MONTH))
            
                Last Month Transactions = CALCULATE([Total transactions], DATEADD('Calendar'[date], -1, MONTH))
            
                Last Year Transactions = CALCULATE([Total transactions], DATEADD('Calendar'[date], -1, YEAR))
            
                Last Year Transactions % difference from Total transactions = 
                VAR __BASELINE_VALUE = [Total transactions]
                VAR __VALUE_TO_COMPARE = [Last Year Transactions]
                RETURN
                    IF(
                        NOT ISBLANK(__VALUE_TO_COMPARE),
                        DIVIDE(__VALUE_TO_COMPARE - __BASELINE_VALUE, __BASELINE_VALUE)
                    )
            
                Number of Stores with Profit < 6k = 
                CALCULATE(
                    COUNTROWS('Stores'),
                    FILTER(
                        ALLSELECTED('Stores'),
                        [Total Profit] < 6000
                    )
                )
            
                Products under 5% profit = 
                COUNTROWS(
                    FILTER(
                        VALUES(Products[product_name]),
                        CALCULATE(DIVIDE([Total Profit], [Total Revenue], 0)) < 0.05
                    )
                )
            
                Quantity returned = SUM(Returns_data[quantity])
            
                Quantity sold = SUM(Transactions_data[quantity])
            
                Rate Weekend Transactions = Divide([Weekend Transactions], [Total transactions])
            
                Return Rate = DIVIDE([Quantity returned], [Quantity sold])
            
                Returns Count = COUNT(Returns_data[return_date])
            
                Revenue Target = [Last Month Revenue] * 1.05
            
                ROI = DIVIDE([Total Profit], [Total Revenue])
            
                Total Cost = SUMX(Transactions_data, Transactions_data[quantity] * RELATED(Products[product_cost]))
            
                Total Profit = [Total Revenue] - [Total Cost]
            
                Total Returns = COUNT(Returns_data[return_date])
            
                Total Revenue = SUMX(Transactions_data, Transactions_data[quantity] * RELATED(Products[product_retail_price]))
            
                Total transactions = COUNT(Transactions_data[transaction_date])
            
                TotalCustomers = COUNT(Customers[customer_id])
            
                TotalCustomersLastYear = 
                CALCULATE(
                    COUNT(Customers[customer_id]),
                    Calendar[Year] = YEAR(TODAY()) - 1
                )
            
                TotalCustomersThisYear = 
                CALCULATE(
                    COUNT(Customers[customer_id]),
                    'Calendar'[date] = YEAR(TODAY())
                )
            
                TotalDistinctCustomers = DISTINCTCOUNT(Customers[customer_id])
            
                Transactions Count = COUNT(Transactions_data[transaction_date])
            
                Unique Products = DISTINCTCOUNT(Products[product_id])
            
                Weekend Transactions = CALCULATE([Total transactions], 'Calendar'[Weekend] = "Y")
            
                YTD Revenue = CALCULATE([Total Revenue], DATESYTD('Calendar'[date]))
