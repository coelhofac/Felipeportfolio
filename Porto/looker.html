Loan Fintech: Realtime Performance Monitoring

Project made with Google's BigQuery (SQL) and Looker Studio as a Capstone Project for the Google Cloud Data Analytics Professional Certificate.

Results

In this project, i've worked in a dataset that covered the loans and customers data of a fintech that had the goal of lending. We've elaborated and interactive dashboard that could be updated regularly through the google storage database.



BigQuery

This dataset had, for our purposes, an almost ready data. What needed some prep was done in the custom query itself 

                    
                        /* Custom BigQuery */
            
                        SELECT 
                            loan.loan_id,
                            loan.customer_id,
                            loan.loan_amount,
                            loan.state,
                            loan.status,
                            loan.purpose,
                            loan.pymnt_plan,
                            PARSE_DATE('%b %Y', loan.issue_date) AS issue_date_converted,
                            INITCAP(REPLACE(loan.purpose, '_', ' ')) AS formatted_purpose,
                            CASE 
                                WHEN customer.annual_inc <= 50000 THEN 'Low Income'
                                WHEN customer.annual_inc BETWEEN 50001 AND 100000 THEN 'Lower-Middle Income'
                                WHEN customer.annual_inc BETWEEN 100001 AND 150000 THEN 'Middle Income'
                                WHEN customer.annual_inc BETWEEN 150001 AND 200000 THEN 'Upper-Middle Income'
                                WHEN customer.annual_inc BETWEEN 200001 AND 300000 THEN 'Affluent'
                                WHEN customer.annual_inc BETWEEN 300001 AND 400000 THEN 'High Income'
                                WHEN customer.annual_inc BETWEEN 400001 AND 500000 THEN 'Very High Income'
                                ELSE 'Income Out of Range'
                            END AS CustomerCategory
                        FROM 
                            `lookfintech.loan` AS loan
                        FULL OUTER JOIN 
                            `lookfintech.customer` AS customer
                        ON 
                            loan.customer_id = customer.customer_id;
