Adventure Works: Tracking a store's sale performance

Project made entirely in PowerBi, with PowerQuery, DAX, and Visualization Tools.

Results

Our goal in this project was retrieve an interactive dataset that allowed our stakeholders to control the performance of their store in a glance. For this particular project i've used all Power Bi for Data Cleaning, Preparing, Exploration and Visualization. Following, you can see the dashboard and then our process.



Importing and cleaning

I've choosed to stablish all the connections between the tables in the BI, what is more appropriate for bigger datasets
Our cleaning work can be summed up in: 1. Verifying and correcting each column's data type 2. Extracting the name of the month of the dates. 3. Identifying and correcting errors through column profiling Correcting capitalization and trimming extra spaces.


Stablishing relationships and creating measures

In this step we've focused on creating our data relationships, building the SCHEMA, stablishing primary and foreign keys, creating all the needed measures for data exploration and visualization based on PRIMARY keys.
